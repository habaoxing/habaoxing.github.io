    权限与实名认证

    哈宝星

    操作场景

    在首次创建静态网站前，您必须先完成授权操作和实名认证，以获得对腾讯云云产品的操作权限。以下将介绍这两项工作的具体操作步骤。

    持续部署权限配置

    您所在的 CODING 用户组需拥有持续部署的管理、部署、删除权限，否则将无法进行网站部署。您可联系您的团队所有者/管理员为您开通这些权限，团队所有者/管理员的操作步骤如下：

    1. 登录 CODING 控制台；

    2. 进入需进行静态网站服务部署的项目；

    3. 单击左下角的「项目设置」；

    1. 确认所在的用户组勾选了静态网站的三个权限；
    1. 若持续部署相关权限为不可选状态，则需前往【团队管理】->【权限配置】中为用户组勾选【部署管理】；

    腾讯云权限配置

    使用 CODING 静态网站服务前，您绑定的腾讯云账号必须拥有 CODING_QCSRole 和 SLS_QCSRole 两个权限角色及这些角色关联的权限策略，否则将无法进行网站部署，您可以在腾讯云访问管理控制台查询是否拥有该权限角色。为方便操作，您可以在 CODING 一键完成授权。授权步骤如下：

    1. 点击静态网站部署按钮，进行部署操作前将自动校验腾讯云权限；
